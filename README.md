# 💰 CarteiraInteligente

> Solução desenvolvida para o **Hackathon ONE G9 | Alura + Oracle**, com o objetivo de transformar dados financeiros em informações inteligentes para apoiar a tomada de decisão.

---

## 📖 Sobre o Projeto

O **CarteiraInteligente** é uma aplicação baseada em Inteligência Artificial que analisa o comportamento financeiro de um usuário a partir de suas transações e indicadores financeiros.

A solução automatiza a classificação de despesas, identifica padrões de consumo, avalia o perfil financeiro do usuário e gera recomendações personalizadas para melhorar sua saúde financeira.

O projeto foi desenvolvido como um MVP (Minimum Viable Product), utilizando técnicas de Ciência de Dados, Machine Learning, APIs REST e serviços da Oracle Cloud Infrastructure (OCI).

---

## 🎯 Objetivos

A aplicação permite:

- Classificar automaticamente transações financeiras;
- Identificar padrões de consumo;
- Avaliar o perfil financeiro do usuário;
- Gerar indicadores de saúde financeira;
- Produzir recomendações personalizadas;
- Disponibilizar os resultados através de uma API REST;
- Utilizar serviços da Oracle Cloud Infrastructure (OCI).

---

## 🚀 Funcionalidades

### 📌 Classificação Inteligente de Transações

Classificação automática das despesas em categorias como:

- 🍽️ Alimentação
- 🚗 Transporte
- 🏠 Moradia
- ❤️ Saúde
- 🎓 Educação
- 🎮 Lazer
- 📱 Serviços
- 🛒 Compras
- 📦 Outros

---

### 📊 Análise do Perfil Financeiro

Com base nas informações fornecidas, o sistema classifica o usuário em um dos seguintes perfis:

- 🟢 Saudável
- 🟡 Em Observação
- 🔴 Em Risco
